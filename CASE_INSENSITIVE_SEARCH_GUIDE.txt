MOVIE NAME SEARCH (searchMovieByName):
????????????????????????????????????????????????????????????????????????????
1. User enters search term (any case)
2. System converts search term to lowercase
3. System converts each movie name to lowercase
4. System compares lowercase strings
5. Original movie names displayed in results

CODE:
    std::string lowerSearchTerm = searchTerm;
    std::transform(lowerSearchTerm.begin(), lowerSearchTerm.end(), 
                   lowerSearchTerm.begin(), ::tolower);
    
    std::string movieName = movies[i].getName();
    std::transform(movieName.begin(), movieName.end(), 
                   movieName.begin(), ::tolower);
    
    if (movieName.find(lowerSearchTerm) != std::string::npos) {
        // Match found!
    }


LANGUAGE FILTER (isLanguage):
????????????????????????????????????????????????????????????????????????????
1. User enters language (any case)
2. System converts both strings to lowercase
3. System compares lowercase strings
4. Original language names displayed in results

CODE (in Movie.cpp):
    std::string lowerLang1 = language;
    std::string lowerLang2 = lang;
    
    for (size_t i = 0; i < lowerLang1.length(); i++) {
        lowerLang1[i] = tolower(lowerLang1[i]);
    }
    for (size_t i = 0; i < lowerLang2.length(); i++) {
        lowerLang2[i] = tolower(lowerLang2[i]);
    }
    
    return lowerLang1 == lowerLang2;

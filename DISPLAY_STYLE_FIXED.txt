?? HOW IT WORKS
???????????????????????????????????????????????????????????????????????????????

STATIC MEMBER PATTERN:
????????????????????????????????????????????????????????????????????????????

1. ONE SETTING FOR ALL INSTANCES:
   • Movie::displayStyle is static
   • Shared by ALL Movie objects
   • Change once, affects everywhere

2. ACCESS FROM ANYWHERE:
   • Movie::setDisplayStyle(2) sets style to Blocks
   • Movie::getDisplayStyle() returns current style
   • No need to pass parameters around

3. AUTOMATIC APPLICATION:
   • displayInfo() checks displayStyle
   • Always uses current global setting
   • Instant effect across all displays

EXAMPLE:
????????????????????????????????????????????????????????????????????????????
    Movie movie1("Inception", 1, 2010, "English", 8.8);
    Movie movie2("Parasite", 2, 2019, "Korean", 8.5);
    
    // Both use default Stars/Bars
    movie1.displayInfo();  // [********--] 8.8/10
    movie2.displayInfo();  // [********--] 8.5/10
    
    // Change global style
    Movie::setDisplayStyle(1);  // Blocks
    
    // Both now use Blocks!
    movie1.displayInfo();  // [########..] 8.8/10
    movie2.displayInfo();  // [########..] 8.5/10

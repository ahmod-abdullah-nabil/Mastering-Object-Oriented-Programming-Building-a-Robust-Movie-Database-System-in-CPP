? KEY CLASSES                                                                  ?
????????????????????????????????????????????????????????????????????????????????

Movie Class:
  Private Members:
    - string name      (Movie title)
    - int id           (Unique identifier)
    - int year         (Release year)
    - string language  (Primary language)
    - int rating       (1-5 scale)
  
  Public Methods:
    - getName(), getId(), getYear(), getLanguage(), getRating()
    - setName(), setId(), setYear(), setLanguage(), setRating()
    - displayInfo()    (Formatted output with stars)
    - isLanguage()     (Language comparison)

MovieDatabase Class:
  Private Members:
    - Movie movies[20] (Array of movies)
    - int movieCount   (Current count)
  
  Public Methods:
    - addMovie()                    (Add movie to database)
    - displayAllMovies()            (Show all movies)
    - displayTopRatedMovies()       (Show highest-rated)
    - displayMoviesByLanguage()     (Filter by language)
    - displayLatestMovies()         (Show most recent)
    - initializeSampleData()        (Load 20 movies)

? OOP CONCEPTS DEMONSTRATED                                                    ?
????????????????????????????????????????????????????????????????????????????????

? ENCAPSULATION
  - Private data members with public getters/setters
  - Data validation (rating 1-5)
  - Controlled access to internal state

? ABSTRACTION
  - Complex algorithms hidden behind simple method calls
  - Users don't see implementation details
  - High-level interface for database operations

? COMPOSITION
  - MovieDatabase "has-a" array of Movie objects
  - One class contains objects of another class

? CONSTRUCTORS
  - Default constructor: Movie()
  - Parameterized constructor: Movie(name, id, year, lang, rating)

? CONST CORRECTNESS
  - const methods: getName() const
  - Guarantees no object modification

? ALGORITHMS USED                                                              ?
????????????????????????????????????????????????????????????????????????????????

Top-Rated Movies:
  1. Find maximum rating (O(n))
  2. Display all movies with max rating (O(n))
  Total: O(n) time complexity

Language Filter:
  - Linear search with string comparison (O(n))

Latest Movies:
  1. Find maximum year (O(n))
  2. Display all movies from that year (O(n))
  Total: O(n) time complexity

? COMMON QUESTIONS & ANSWERS                                                   ?
????????????????????????????????????????????????????????????????????????????????

Q: Why array instead of vector?
A: Demonstrates fundamental C++ and manual memory management.

Q: How to add more movies?
A: Change MAX_MOVIES constant or refactor to dynamic allocation.

Q: What if no movies match filter?
A: Program displays "No movies found" message with count 0.

Q: Why separate .h and .cpp files?
A: Separation of interface and implementation - best practice.

Q: What about duplicate ratings/years?
A: System handles multiple results correctly (arrays/loops).
